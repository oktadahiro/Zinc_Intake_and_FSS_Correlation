# 亜鉛摂取と機能性身体症候群の関係（ダミーデータによる）

## 仮説
亜鉛の欠乏症は皮膚炎、味覚障害、慢性下痢、免疫機能障害、成長遅延、生鮮発育障害などの身体的なものがよく知られている。  
また、亜鉛の欠乏はメラトニンやセロトニンの合成を妨げることがわかっており、精神的な疾患との相関も示唆されている。
今回はダミーデータを用い亜鉛の摂取とFSS(機能性身体症候群)の線形性などを評価した。

### ダミーデータの特徴
- サンプルサイズ: n=1000（メタボ群500、健常群500）
- 亜鉛摂取量: メタボ群7.5±2.0mg/日、健常群9.5±2.5mg/日
- FSS: メタボ群38±8点、健常群24±6点（範囲9-63点）
- 想定相関係数: r ≈ -0.35（負の相関）
- 交絡因子: 性別、年齢、BMI、運動習慣、睡眠時間、地域

### ダミーデータの変数説明
- Group: Metabolic（メタボ群）/ Healthy（健常群）
- Exercise: 0=運動なし、1=週1-2回、2=週3回以上
- Region: 1=都市部、2=地方、3=農村部
- FSS_Score: 36点以上で病的疲労の可能性
